Group Weather

Online Weather Application utilizing Crowdsourcing Principles

This application was an integration of crowdsourcing and weather data that sought to provide more reliable weather information. Eye Witnesses use the simple front end landing page to confirm certain weather behaviors at their location. This feedback is fed into the database and algorithmically combined with predicted weather data to provide a more real time and accurate weather outlook.

    Technologies Utilized:
    Bootstrap
    Ajax
    Vue.JS
    PHP
    MySQL
    ipinfo.io
    openweather.com

  • Date: Spring 2017
  • Course: Global Information Systems

Sports Enterprise Playoff System

A Scalable Sports Bracket System

This application is a scalable sports system that managed a binary playoff bracket for each respective sport. The system was required to have an optimized database component as well as a team developed list of functional dependencies.
